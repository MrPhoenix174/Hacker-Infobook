


# **Платформы для обучения**
### [**TryHackMe:**](https://tryhackme.com/) Платформа для обучения Иб, есть "пути обучения", большое количество практических комнат (есть для всех направлений кб). Подойдет как для новичков, так и для опытных.
<br>

### [**Hack The Box:**](https://www.hackthebox.com/) Онлайн платформа для практики кибербезопасности(для новичков и продвинутых). HTB Academy - образовательный портал с пошаговыми курсами для обучения с нуля и подготовки к сертификации.

<br>

### [**PortSwigger Academy:**](https://portswigger.net/web-security) Бесплатная онлайн-платформа от создателей Burp Suite для обучения веб-безопасности. 

<br>

### [**Root Me:**](https://www.root-me.org/) Популярная платформа, предлагающая более 500 практических заданий и виртуальных сред. Ориентирована на решение CTF задач на самые разные темы.

<br>

### [**HackerLab:**](https://hackerlab.pro/) Одна из крупнейших в СНГ практических онлайн-платформ для обучения, тренировки и сертификации специалистов по информационной безопасности.
