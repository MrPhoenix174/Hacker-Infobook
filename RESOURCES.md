# Полезные ресурсы и инструменты

Здесь собраны ссылки на обучающие платформы, сторонние библиотеки и инструменты, которые помогают в учебе и практике.

---

## 1. Платформы для практики 


* **[TryHackMe](https://tryhackme.com/)** - Идеально для старта. Есть пошаговые Roadmap обучения для новичков.
* **[Hack The Box](https://www.hackthebox.com/)** - Более серьезные машины для пентеста и подготовки к OSCP.
* **[PortSwigger Academy](https://portswigger.net/web-security)** - Топовый ресурс для изучения Web безопасности от создателей Burp Suite.
* **[Root-Me](https://www.root-me.org/)** - Огромная база тасков по реверсу, криптографии и вебу.

## 3. Инструментарий (Must Have)
*Полезные сервисы.*

* **[CyberChef](https://gchq.github.io/CyberChef/)** - "Швейцарский нож" для дешифровки, кодирования и анализа данных.
* **[GTFOBins](https://gtfobins.github.io/)** - Список бинарников Linux, которые можно использовать для повышения привилегий (PrivEsc).
* **[Exploit-DB](https://www.exploit-db.com/)** - База данных публичных эксплойтов.
